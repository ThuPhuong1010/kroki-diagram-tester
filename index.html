How to use Confluence Sync
🌐 Preview Mode (GitHub Pages)
1
Draw your diagram in the editor — live preview renders via Kroki.
2
Copy the URL from the URL bar (bottom) and embed it in Confluence via Insert → Image → URL.
ℹ️ Direct sync is disabled on GitHub Pages due to browser CORS restrictions. Use Full Sync mode below.
🖥️ Full Sync Mode (local server)
1
Start local server — in your terminal:
cd D:\MVillage && npm run kroki
2
Open http://localhost:3333 in your browser. Credentials are loaded from .env automatically.
3
Pick a page via Browse — select a Space then a Page. Page ID fills in automatically.
4
Sync — click Sync to Confluence. PNG uploads as an attachment. Embed once, it updates in-place on re-sync.
💡 Python alternative: python sync_to_confluence.py diagrams/flow.mmd — same result from terminal.
